# SolidObserver
### *Observe your Solid Stack like a pro!*

SolidObserver is a production-grade observability solution for Rails 8's Solid Stack (Solid Queue, Solid Cache, Solid Cable). It provides unified monitoring, health metrics, and debugging tools across all three Solid technologies.
